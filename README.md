📺 CMTV - Tecnologia Street Power
Este é o repositório do site da CMTV, uma plataforma que desenvolvi com foco em entretenimento digital (IPTV). A ideia aqui foi criar uma interface "Premium" que não apenas apresentasse os serviços, mas que fosse extremamente rápida e fácil de navegar tanto no PC quanto no celular.

📌 O que o projeto entrega
Criei este site para centralizar tudo o que o cliente precisa em um só lugar, focando em converter o visitante em usuário.

Design Dark & Moderno: Usei uma paleta Dark com detalhes em vermelho para dar aquele ar de tecnologia e exclusividade.

Foco em Conversão: Adicionei botões estratégicos de "Liberar Acesso" e um botão flutuante de WhatsApp que facilita o suporte imediato.

Central de Apps: Montei uma seção específica para facilitar o download e instalação em Android, Smart TVs, PC e iOS.

Navegação Fluida: Menus dropdown organizados para planos e tutoriais, evitando poluição visual.

🛠️ Tecnologias que utilizei
Optei por manter o código limpo e leve, garantindo uma boa performance:

HTML5: Estrutura semântica para melhor organização.

CSS3: Layouts baseados em Flexbox e Grid, além de variáveis (:root) para facilitar a manutenção das cores.

JavaScript: Lógica para interatividade dos menus e automação das chamadas de chat.

Google Fonts: Tipografia moderna (Archivo Black e Inter) para passar autoridade no design.

📂 Organização das Pastas
O projeto está estruturado da seguinte forma:

├── public/

│   ├── images/ # Logos e ícones dos sistemas

│   ├── index.html   # Home principal do site

│   ├── style.css    # Toda a estilização e responsividade
